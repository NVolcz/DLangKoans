DLangKoans
==========

inspired from RubyKoans, CppKoans, GoKoans; an educational unit test suite for the [D Programming Language](http://dlang.org)

work in progress


How it works:
------------
 - run rdmd main.d
 - observe test errors
 - think and make the test pass
 - goto first step
